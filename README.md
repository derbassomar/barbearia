# barbearia
💈Projeto: berber shoEste projeto é um website temático para uma barbearia, desenvolvido com HTML, CSS e JavaScript. Composto por 4 páginas interativas e vinculadas, o site foi criado com o objetivo de proporcionar uma experiência elegante e moderna, destacando o conceito de cuidado pessoal masculina,

<h1>🖥️ Funcionalidades e Características</h1>
Design Responsivo: Adaptado para diferentes tamanhos de tela, garantindo uma boa usabilidade em dispositivos móveis e desktops.
Navegação Interativa: Menu superior que facilita a navegação entre as páginas principais: Início, Serviços, Sobre e Contato.
Seção de Sobre Nós: Uma introdução à filosofia e aos valores da barbearia, reforçando a tradição e o toque contemporâneo dos serviços.
Galeria de Imagens: Exibição visual dos serviços e do ambiente da barbearia, proporcionando uma visão clara do espaço e dos cuidados oferecidos.
Informações de Contato e Localização: Seção com endereço, horário de funcionamento e detalhes para agendamento, otimizando a comunicação com os clientes.

<h1>🌐 Tecnologias Utilizadas</h1>
HTML5: Estrutura semântica do site.
CSS3: Estilização e efeitos visuais, incluindo temas em preto e branco e uso de transições.
JavaScript: Funções interativas, como a galeria de imagens e a navegação entre páginas.
